Fluxos e Ciclos de Trabalho Padronizados (Requisito U02)

Todas as ações críticas do CRM seguem obrigatoriamente a sequência operacional estruturada:

Entrada → Pré-condição → Ação Principal → Processamento → Resultado Visual → Próximo Passo

F1 - Acesso e Abertura (Autenticação)

Entrada: Abertura do App / Login

Pré-condição: Credenciais válidas disponíveis no aparelho

Ação: Preenchimento e submissão com e-mail e senha

Processamento: "Verificando..." + banner de status

Resultado: Abertura do Painel com carregamento assíncrono

Próximo Passo: Radar / Início operacional

F4 - Conversas e Atendimento

Entrada: Aba Conversas

Pré-condição: Conexão ativa com o integrador de WhatsApp

Ação: Seleção de contato e redação da resposta

Processamento: "Enviando..." + feedback do botão

Resultado: Histórico atualizado localmente

Próximo Passo: Retorno à listagem de mensagens ativas

F5 - Comando Assistido por Voz

Entrada: Botão "Capturar voz" no Radar

Pré-condição: Permissão de gravação ativa

Ação: Gravar comando em áudio e parar

Processamento: Transcrevendo e processando com modelo de IA

Resultado: Filtro automático ou ação aplicada

Próximo Passo: Confirmação de êxito na barra lateral

Matriz de Tradução e Desmistificação de Termos (Requisito U01)

Tabela de tradução obrigatória para garantir uma interface livre de vocabulário puramente sistêmico ou de programação:

Termo Técnico (Anterior) Tradução Operacional (Nova) Contexto de Aplicação / Motivo Técnico
CMS / CMS Compartilhado Base de dados da imobiliária / Painel O termo "CMS" confunde o usuário final. Usar "Base de dados" ou "Painel" aproxima da realidade comercial.
OKF / OKF Tree Árvore / Central de Conhecimento "OKF" refere-se à estrutura de conhecimento de arquivos markdown. Traduzido para "Árvore de Conhecimento".
Bootstrap / Runtime Local Inicialização / Sistema local Indica o carregamento inicial da aplicação e sincronização nativa. Exposto agora como carregamento simples.
Bridge / Bridge WhatsApp Conexão do WhatsApp / Integrador O integrador de mensagens não deve parecer uma "ponte" sistêmica, mas sim uma "conexão" funcional direta.
Grounding / Grounding Sources Fontes de Informação / Conhecimento A ancoragem semântica da IA de mercado agora é exposta como fontes de informação revisáveis por humanos.
Payload / Session / File Service Dados / Conversa / Gerenciador de Arquivos Remoção de referências brutas de APIs e estruturas do Capacitor/Android na camada visual.
